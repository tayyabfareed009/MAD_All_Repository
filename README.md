# MAD Repository - Collection of Projects

This repository contains all major MAD course projects, including fully functional **Expo apps**, backend setups (Node.js), and working snapshots/videos. Each project folder contains the complete code and instructions to run locally.

---

## Projects Included

1. **Skill Swap - Assignment 1**  
   - **Description:** An app for users to exchange skills.  
   - **Stack:** React Native (Expo), Node.js (optional backend).  
   - **Commands to Run:**
     ```bash
     npx expo start
     ```
   - **Features:** User signup/login, skill listing, messaging.  
   - **Snapshots/Video:** Included in the project folder.

2. **Quiz 01 & ToDo List App**  
   - **Description:** Interactive quiz app and task management app.  
   - **Stack:** React Native (Expo), Node.js backend (optional).  
   - **Commands to Run:**
     ```bash
     npx expo start
     node server.js   # if backend exists
     ```
   - **Features:** Multiple choice questions, score tracking, task CRUD functionality.  
   - **Snapshots/Video:** Included in the project folder.

3. **Assignment 02 - Ecommerce App**  
   - **Description:** Full working Ecommerce mobile app.  
   - **Stack:** React Native (Expo), Node.js backend, MongoDB (if used).  
   - **Commands to Run:**
     ```bash
     npx expo start
     node server.js
     ```
   - **Features:** Product listing, cart functionality, search, admin panel.  
   - **Snapshots/Video:** Included in the project folder.

4. **MAD Mid Exam - Coffee Shop**  
   - **Description:** MID/MAD Exam project simulating a coffee shop app.  
   - **Stack:** React Native (Expo), Node.js backend.  
   - **Commands to Run:**
     ```bash
     npx expo start
     node server.js
     ```
   - **Features:** Menu browsing, cart, order management.  
   - **Snapshots/Video:** Included in the project folder.

---

## General Commands for Expo Projects

- **Create a new Expo app:**

npx create-expo-app mynewapp
# Start an existing Expo app:
 cd project-folder 
 npx expo start
Run backend server (Node.js):


node server.js
